# en
Some websites prevent you from right-clicking, selecting text, or copying content. Click & Copy removes these restrictions so you can use the web normally.

How it works:
- Click the extension icon to enable it for the current site.
- A green checkmark badge shows when the extension is active.
- Right-click the icon for more options: enable globally, per site, or per page.

Features:
- Restores right-click context menu.
- Restores text selection.
- Restores copy and paste.
- 3-level control: global, per-site, and per-page.
- Works in incognito without saving any data.

Click & Copy does not collect, transmit, or share any user data. All settings are stored locally on your device.
